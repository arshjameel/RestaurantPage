## Restaurant Page
A fully dynamic restaurant website built with vanilla JavaScript, HTML, and CSS. 
This project demonstrates **tabbed navigation with modular DOM manipulation**. 
All content loads via JS without page refreshes.

### Features
- Dynamic tab switching (Home, Menu, Contact)
- Modular page components (`home.js`, `menu.js`, `contact.js`)
- Webpack bundling with image imports
- Responsive design with flexbox layout

### Tech Stack
- Vanilla JavaScript (ES6 modules)
- Webpack 5 (with asset handling)
- CSS Flexbox & Grid
- Responsive images
